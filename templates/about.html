About the Project

Project Overview

The Machine Learning Based Phishing URL Detection System is designed to protect users from phishing attacks by analyzing website URLs and identifying potentially malicious sites. Using advanced machine learning algorithms, the system examines various URL characteristics to determine the likelihood of a website being a phishing attempt.

Objectives

  • Detect phishing URLs with high accuracy using machine learning
  • Provide real-time security warnings to users
  • Increase awareness of cybersecurity threats
  • Offer a simple, user-friendly web interface

How It Works

1

URL Input

User enters a website URL into the system

2

Feature Extraction

System extracts 35+ features from the URL including length, special characters, patterns, and structure

3

ML Prediction

Random Forest model analyzes features and predicts whether URL is phishing or legitimate

4

Result Display

User receives instant result with confidence score and safety recommendations

Dataset & Model

The system uses a comprehensive phishing URL dataset containing:

  • 11,430 URLs with balanced phishing and legitimate samples
  • 87 extracted features per URL
  • Random Forest classifier with 96% accuracy
  • Features include URL structure, syntax patterns, and content analysis

System Architecture

The application follows a three-tier architecture:

Presentation Layer

HTML, CSS, JavaScript

Application Layer

Flask (Python)

ML Detection Engine

Scikit-learn, Random Forest

Cybersecurity Awareness

Phishing attacks are one of the most common cybersecurity threats where attackers create fake websites that mimic legitimate ones to steal sensitive information. This project aims to help users identify potentially malicious URLs before visiting them, making the internet a safer place.
